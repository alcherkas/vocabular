# Decisions Pending

Agents write here when they need human input before proceeding.

## How it works
1. Agent writes an entry below with options.
2. Agent sets goal status to `[blocked: decision-pending]` in `GOALS.md` and stops.
3. Human reads the entry, picks an option by adding `**Choice: X**` to the entry.
4. Agent resumes, applies the choice, moves the resolved entry to `docs/decisions-log.md`.
